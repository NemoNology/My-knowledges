#SQL
# Определение
Structured Query Language - Язык структурированных запросов
# Основные типы данных
## Строка
### char(n)
n - количество символов в строке. `char` заполняет все пустые символы пробелами.
```sql
char([n])
char(5) от 'Ухо' = 'Ухо  '
```
### varchar(n)
```sql
varchar([n])
```
## Число
### number(точность, маштаб)
```sql
number()
```
